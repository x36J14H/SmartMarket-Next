# Personal API (Личный кабинет)

Base URL: `http://<host>:<port>/smartmarket/hs/personal`

Управление профилем, избранным и корзиной пользователя.  
Все эндпоинты требуют заголовок `X-Auth-Token`.

---

## PATCH /profile

Обновление данных аккаунта. Все поля необязательные — передавай только то, что нужно изменить.

### Заголовки

| Заголовок      | Значение |
|----------------|----------|
| `X-Auth-Token` | Активный токен |

### Тело запроса

```json
{
  "name": "Иван Иванов",
  "phone": "+79001234567",
  "delivery_address": "г. Москва, ул. Ленина, д. 1, кв. 10"
}
```

| Поле               | Тип    | Описание |
|--------------------|--------|----------|
| `name`             | string | Отображаемое имя |
| `phone`            | string | Номер телефона (без подтверждения) |
| `delivery_address` | string | Адрес доставки |

### Ответ `200`

```json
{ "ok": true }
```

### Ответ `401`

```json
{ "error": "Токен не передан" }
```

## GET /wishlist

Список избранных товаров текущего пользователя. Отсортирован по дате добавления (новые первые).

### Заголовки

| Заголовок      | Значение |
|----------------|----------|
| `X-Auth-Token` | Активный токен |

### Ответ `200`

```json
[
  {
    "id": "fe0a06db-288a-11f1-8d5c-4c2338935cb2",
    "name": "iPhone 15 Pro",
    "article": "IPH15PRO",
    "slug": "iphone-15-pro",
    "price": 89990.00,
    "imageUrl": "5b270d7e-2922-11f1-8d5d-4c2338935cb2"
  }
]
```

### Ответ `401`

```json
{ "error": "Токен не передан" }
```

## GET /cart

Содержимое корзины текущего пользователя. Отсортировано по дате добавления (старые первые).

### Заголовки

| Заголовок      | Значение |
|----------------|----------|
| `X-Auth-Token` | Активный токен |

### Ответ `200`

```json
[
  {
    "id": "fe0a06db-288a-11f1-8d5c-4c2338935cb2",
    "name": "iPhone 15 Pro",
    "article": "IPH15PRO",
    "slug": "iphone-15-pro",
    "price": 89990.00,
    "imageUrl": "5b270d7e-2922-11f1-8d5d-4c2338935cb2",
    "qty": 2
  }
]
```

| Поле       | Тип    | Описание |
|------------|--------|----------|
| `id`       | string | UUID товара |
| `name`     | string | Наименование |
| `article`  | string | Артикул |
| `slug`     | string | URL-slug |
| `price`    | number | Актуальная цена |
| `imageUrl` | string | fileId главного изображения |
| `qty`      | number | Количество в корзине |

---

## POST /cart/{id}

Добавить товар в корзину. Если товар уже есть — количество увеличивается на `qty`.

### Path-параметры

| Параметр | Тип    | Описание |
|----------|--------|----------|
| `id`     | string | UUID товара |

### Заголовки

| Заголовок      | Значение |
|----------------|----------|
| `X-Auth-Token` | Активный токен |

### Тело запроса (необязательно)

```json
{ "qty": 1 }
```

Если тело не передано или `qty` не указан — добавляется 1 единица. `qty` должен быть больше 0.

### Ответ `200`

```json
{ "ok": true }
```

### Ответ `400`

```json
{ "error": "qty должен быть больше 0" }
```

### Ответ `404`

```json
{ "error": "Товар не найден" }
```

---

## PATCH /cart/{id}

Установить точное количество товара в корзине. Если `qty` = 0 — позиция удаляется.

### Path-параметры

| Параметр | Тип    | Описание |
|----------|--------|----------|
| `id`     | string | UUID товара |

### Заголовки

| Заголовок      | Значение |
|----------------|----------|
| `X-Auth-Token` | Активный токен |

### Тело запроса

```json
{ "qty": 3 }
```

`qty` должен быть >= 0. При `qty` = 0 позиция удаляется из корзины.

### Ответ `200`

```json
{ "ok": true }
```

### Ответ `400`

```json
{ "error": "qty не может быть отрицательным" }
```

### Ответ `404`

```json
{ "error": "Товар не найден в корзине" }
```

## Коды ответов

| Код | Описание |
|-----|----------|
| `200` | Успех |
| `400` | Ошибка валидации (отрицательный qty и т.п.) |
| `401` | Токен не передан или недействителен |
| `404` | Товар не найден |

## Примеры запросов

```
# Получить избранное
GET /wishlist
X-Auth-Token: <token>

# Добавить в избранное
POST /wishlist/fe0a06db-288a-11f1-8d5c-4c2338935cb2
X-Auth-Token: <token>

# Удалить из избранного
DELETE /wishlist/fe0a06db-288a-11f1-8d5c-4c2338935cb2
X-Auth-Token: <token>

# Получить корзину
GET /cart
X-Auth-Token: <token>

# Добавить в корзину (qty=1 по умолчанию)
POST /cart/fe0a06db-288a-11f1-8d5c-4c2338935cb2
X-Auth-Token: <token>

# Добавить в корзину с количеством
POST /cart/fe0a06db-288a-11f1-8d5c-4c2338935cb2
X-Auth-Token: <token>
Content-Type: application/json
{"qty": 3}

# Изменить количество
PATCH /cart/fe0a06db-288a-11f1-8d5c-4c2338935cb2
X-Auth-Token: <token>
Content-Type: application/json
{"qty": 5}

# Удалить позицию
DELETE /cart/fe0a06db-288a-11f1-8d5c-4c2338935cb2
X-Auth-Token: <token>

# Очистить корзину
DELETE /cart
X-Auth-Token: <token>
```
